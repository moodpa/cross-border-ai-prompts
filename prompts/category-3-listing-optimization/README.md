## Category 3: Listing Optimization (7 prompts)
### Prompt 15: Amazon SEO Title Generator
```
Generate 5 optimized Amazon product titles for [product name] in [category].
Each title must:
- Start with the primary keyword
- Include 3-5 key features/benefits
- Be under 200 characters
- Sound natural, not keyword-stuffed
- Follow Amazon's style guide for [category]
For each title, explain the keyword strategy behind it.
```
### Prompt 16: Bullet Point Converter
```
Transform this product description into 5 compelling Amazon bullet points:
[product description]
Each bullet must:
- Start with a benefit (not just a feature)
- Be under 500 characters
- Include relevant keywords naturally
- Address a specific customer pain point
- End with an emotional trigger (confidence, relief, excitement)
Current description: "[paste product description]"
```
### Prompt 17: A+ Content Copywriter
```
Write Amazon A+ Content (Enhanced Brand Content) for [product].
Sections needed:
1. Hero section headline and sub-headline (lifestyle benefit focus)
2. Feature comparison grid (5 key features vs competitors/standard)
3. "Why Choose Us" narrative (100 words, emotional + rational)
4. Technical specifications section (clean, scannable)
5. Usage/application scenarios (3-4 scenarios)
6. Guarantee/warranty and call-to-action
Tone: confident, educational. Not hype-y. Format as [section: text].
```
### Prompt 18: Etsy SEO Listing Optimizer
```
Create an optimized Etsy listing for [product name].
Include:
- Title: 10-15 relevant keywords, natural flow, full 140 characters
- Tags: 13 tag slots filled with high-value search terms
- Category and subcategory recommendations
- Attributes: fill all relevant attributes
- Description: 3 paragraphs (first: hook + keyword rich, second: details + benefits, third: policies + CTA)
- Price positioning relative to competitors
Explain keyword research methodology used.
```
### Prompt 19: eBay Listing Adapter
```
Convert this Amazon product listing into an eBay-optimized version:
[Amazon listing text or URL]
eBay-specific optimizations:
- Title: 80 characters max, front-load keywords, include condition
- Item specifics: fill all [category]-specific fields
- Description: eBay HTML format, mobile-first layout
- Shipping: highlight your shipping speed and free shipping thresholds
- Returns policy: competitive but protective language
- Best Offer settings recommendation
Note differences in buyer behavior between Amazon and eBay.
```
### Prompt 20: Shopify Product Page Copy
```
Write compelling product page copy for Shopify store selling [product].
Sections:
1. Above-the-fold headline and subheadline (maximum impact)
2. Feature-to-benefit translation table (10 points)
3. "How It Works" or "What's Inside" section (step-by-step)
4. Social proof section framework (what to collect, how to display)
5. FAQ section (top 5 questions with answers)
6. Size/measurement guide if applicable
7. Call-to-action variations (3 options)
Include meta description for SEO (under 160 chars).
```
### Prompt 21: Image Alt Text Generator
```
Generate SEO-optimized alt text for [product's] listing images.
For each image in a typical 7-image listing:
- Main image: describe product, primary keyword, key benefit
- Lifestyle image: setting, usage, emotion conveyed
- Size/dimension image: measurements, scale reference
- Feature image: specific feature highlighted
- Ingredient/material image: quality signals
- Comparison image: what makes it different
- Packaging image: what customer receives
Each alt text: 5-15 words, keyword-rich but natural.
```
