## Category 7: Inventory & Operations (4 prompts)
### Prompt 41: Inventory Reorder Calculator
```
Create a reorder point calculation for [product].
Input variables to define:
- Lead time from supplier (in days)
- Average daily sales (units)
- Safety stock (days of stock to keep as buffer)
- Current stock level
- Optimal order quantity (economic order quantity)
Calculate:
- Reorder point (when to place next order)
- Days until stockout at current sales rate
- Recommended order quantity
- Total inventory cost at reorder
- Cash flow impact
Leave variables as parameters I can adjust.
```
### Prompt 42: Shipping Strategy Optimizer
```
Analyze shipping options for [product] from China to [destination market].
Compare:
- Air freight (express vs economy)
- Sea freight (LCL vs FCL)
- Rail freight (if applicable)
- Consolidated shipping vs direct
For each method:
- Transit time
- Cost per unit (based on product dimensions and weight)
- MOQ or minimum volume requirements
- Customs complexity
- Insurance options
- Tracking capabilities
Recommend the optimal mix for: standard orders, rush orders, bulk replenishment.
```
### Prompt 43: Cash Flow Forecaster
```
Build a 90-day cash flow forecast for my cross-border operation selling [product type].
Input parameters (leave as variables I can fill):
- Monthly revenue forecast
- COGS (cost of goods sold, including shipping)
- Marketplace/platform fees
- Marketing spend
- Operating expenses
- Payment terms (receivables and payables)
Generate:
- Weekly cash position for 90 days
- Peak cash requirement
- Breakeven point analysis
- 3 "what-if" scenarios (best case, base, worst case)
- Days of runway at current burn rate
```
### Prompt 44: Warehousing Decision Framework
```
Help me decide between FBA (Fulfilled by Amazon), FBM (Fulfilled by Merchant), 3PL, or self-warehousing for [product type] selling in [market].
Comparison across:
- Cost per unit (storage + fulfillment)
- Speed to customer
- Return handling
- Inventory control
- Brand experience control
- Scalability
- Minimum commitment
- Integration complexity
Recommend the optimal setup for my scale ([current monthly orders]) and growth trajectory.
```
