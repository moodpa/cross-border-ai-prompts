## Category 4: Copywriting (8 prompts)
### Prompt 22: Facebook Ad Copy Generator
```
Create 5 Facebook ad copy variations for [product] targeting [audience].
Each ad includes:
- Primary text (125-150 characters, hook in first line)
- Headline (40 characters max)
- Description (30 characters max)
- Call-to-action button recommendation
- Emotional trigger (pain point, aspiration, curiosity, urgency)
Also suggest 3 audience targeting angles based on the product's natural fit.
```
### Prompt 23: TikTok Script Writer
```
Write a 30-60 second TikTok script to promote [product].
Structure:
- Hook (first 3 seconds, stop-scroll)
- Problem introduction (5-10 seconds)
- Product reveal (5 seconds)
- Key benefit demonstration (10-15 seconds)
- Social proof or results (5-10 seconds)
- Call-to-action (5 seconds)
Include suggested visual elements, text overlays, and audio style.
```
### Prompt 24: Email Sequence for New Customers
```
Create a 5-email welcome sequence for customers who purchased [product type].
Email 1: Thank you + order confirmation + what to expect
Email 2: Product usage tips + best practices
Email 3: Related products/cross-sell (gentle)
Email 4: Request review + incentive
Email 5: Loyalty program intro + future recommendations
Each email: subject line (40-60 chars), preview text, body (80-120 words), CTA.
Tone: helpful, not salesy. Protocol: build relationship first, sell later.
```
### Prompt 25: Abandoned Cart Recovery
```
Write 3 abandoned cart email variations for a customer who left [product] in their cart ($[price]).
Variation A: Gentle reminder + social proof (reviews, ratings)
Variation B: Scarcity angle (limited stock, price change warning)
Variation C: Incentive offer (free shipping, small discount)
Each: subject line (max 50 chars), body (max 100 words), CTA.
Track which variation performs best and iterate.
```
### Prompt 26: Influencer Outreach DM
```
Draft a direct message to reach out to [type of influencer] for a potential collaboration on [product].
The message should:
- Compliment their specific content (show you researched them)
- Briefly introduce your brand/product
- Propose a specific collaboration idea
- Make it easy for them to say yes (low commitment)
- Include what's in it for them (commission, free product, exclusive access)
Goal: high reply rate. Not generic, not too long. Max 200 words.
Tone: respectful partnership, not "free product for exposure."
```
### Prompt 27: Product Launch Landing Page
```
Write copy for a product launch landing page for [product] targeting [audience].
Sections:
1. Hero: headline (max 20 words) + subheadline (2 sentences) + primary CTA
2. Problem-Agitate-Solution framework (150 words)
3. Features vs benefits table (10 items)
4. Social proof: testimonial framework, review highlights
5. Scarcity/urgency element (time-limited, stock-limited)
6. Risk reversal (guarantee, return policy)
7. Final CTA with urgency
Include meta title and description for SEO.
```
### Prompt 28: Customer Testimonial Collection Script
```
Write a script/email to request testimonials and reviews from recent buyers of [product].
Best practices to include:
- Timing: when to ask (3-7 days after delivery)
- Incentive: what to offer (discount on next purchase, freebie)
- Format options (written, video, star rating)
- Specific questions to guide their testimonial
- Permission to use on website and social media
Draft 3 versions: email, in-package insert card, and DM on social media.
```
### Prompt 29: Competitor Ad Teardown
```
Analyze this competitor's ad: [provide ad text or describe ad].
Break down:
- Emotional trigger being used
- Target audience assumption
- Hook effectiveness
- Offer strength
- CTA clarity
- 3 ways to make a better version
Also suggest: "If this is their angle, what's the counter-angle we should use?"
```
