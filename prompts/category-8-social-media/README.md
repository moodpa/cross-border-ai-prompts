## Category 8: Social Media Marketing (6 prompts)
### Prompt 45: Content Calendar Generator
```
Create a 30-day social media content calendar for a brand selling [product type].
For each day:
- Platform (TikTok, Instagram, X/Twitter, LinkedIn, or Pinterest)
- Content type (video, carousel, text post, story, live)
- Topic/angle (educate, entertain, inspire, convert)
- Key message and hook
- Hashtags (5-10 per post, mixing broad and niche)
- Best posting time
Theme weeks: Education Week, Behind-the-Scenes, Customer Spotlight, Product Deep Dive.
```
### Prompt 46: Viral Angle Creator
```
Generate 20 viral content angles for [product type].
For each angle:
- One-sentence concept
- Emotional trigger (curiosity, surprise, humor, aspiration, fear of missing out)
- Recommended format (short video, image meme, thread, story)
- Why it has viral potential
- Example headline/hook
Priority angles: ones that show transformation, generate debate, or teach something unexpected.
```
### Prompt 47: Instagram Carousel Builder
```
Design a 10-slide Instagram carousel post about [topic related to product/brand].
Slide 1: Cover slide (bold headline, brand colors)
Slide 2-3: The problem (relatable, specific)
Slide 4-5: The solution (product mentioned naturally)
Slide 6-7: How it works (step-by-step, clear visuals)
Slide 8-9: Results/proof (before/after, testimonials, data)
Slide 10: CTA + engagement hook (save, share, comment, shop)
Caption: hook in first line, 3-5 relevant hashtags, CTA at end.
```
### Prompt 48: Pinterest SEO Pin Creator
```
Create Pinterest-optimized content for [product/category].
For each pin:
- Title: keyword-rich, max 100 characters
- Description: 2-3 sentences with keywords, max 500 characters
- Board category recommendation
- Rich Pin type selection (product pin, article pin)
Suggest 5 board names I should create for this product category.
Include seasonal pin ideas for upcoming holidays/events.
```
### Prompt 49: UGC (User Generated Content) Brief
```
Write a creative brief for user-generated content about [product].
What to include in the brief:
- Theme and mood (3-5 reference keywords)
- Key features to highlight
- "Do this, not that" visual guidelines
- Script/talking points (if video)
- Hashtags to include
- Length requirements
- Submission format and deadline
- Incentive structure
- Rights/usage agreement (what we can repost)
Make it easy enough that a non-creator customer can follow it.
```
### Prompt 50: Social Listening Setup
```
Create a social listening framework for tracking mentions of [brand/product/niche].
Keywords to monitor:
- Brand name variations
- Product name and common misspellings
- Industry keywords
- Competitor names
- Common pain points in your niche
For each keyword cluster:
- Best platforms to monitor
- Sentiment analysis approach
- Response priority (urgent, standard, low)
- Response templates for common scenarios
- Weekly reporting format
Track share of voice vs top 3 competitors.
```

## Bonus: Getting Started Guide
### How to use these prompts effectively
1. **Pick one category** that addresses your biggest pain point today
2. **Copy the prompt** and paste it into ChatGPT, Claude, or Gemini
3. **Fill in the [brackets]** with your specific details
4. **Review and customize** the output to match your brand voice
5. **Save working prompts** with your specific data filled in for reuse
### Pro Tips
- For ChatGPT: use a fresh chat for each prompt to avoid context contamination
- For Claude: use Projects to organize prompts by category
- For Gemini: enable "Google Workspace" integration for spreadsheet/email automation
- Combine prompts: use outputs from one prompt as inputs to another
