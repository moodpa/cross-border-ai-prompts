# 🤝 Contributing

Thanks for checking out this prompt pack! Here's how you can contribute:

## 🐛 Found a bug or broken prompt?
Open an issue describing the problem and how to reproduce it.

## 💡 Have a suggestion?
Open an issue with the "enhancement" label. Include:
- What category your prompt belongs to
- The prompt itself
- What problem it solves
- Example output (if possible)

## 📝 Want to add a new prompt?
1. Fork the repo
2. Add your prompt to the relevant category file in `prompts/`
3. Follow the existing format:
   ```
   ### Prompt N: Title
   ```
   ```
   [prompt template with [brackets] for custom inputs]
   ```
4. Submit a PR

## 🗺️ Translations
If you want to translate prompts to another language, create a `prompts/[lang]/` directory (e.g., `prompts/zh/` for Chinese).

## ⭐ Other ways to help
- Star the repo
- Share on social media
- Write about it on Dev.to/Medium
- Buy the pack and tell your friends
